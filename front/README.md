# 検証仕様

ERC721 でアイテムの交換ができるサイト

## アカウント初期値

・ランダムな基本アイテムの付加

## 機能

・metamask ログインできる  
・自身の所持アイテムが見れる  
・他のアカウント一覧が見れる  
・特定のアカウントにトークンの譲与の許可ができる
・他のアカウントにアイテムを渡せる
